### ■ 5. X環境（LXQt）のインストール

#### パッケージアップデート

```sh
apk update
apk upgrade
```

#### LXQt + 通常必要になる追加パッケージ

```sh
apk add lxqt xorg-server xf86-video-fbdev lightdm lightdm-gtk-greeter
```

#### ウィンドウマネージャ（Openbox）

```sh
apk add openbox
```

#### 自動起動設定（ディスプレイマネージャ）

```sh
rc-update add lightdm
```

---

### ■ 6. グラフィック対応

UTM SEでは GPUアクセラレーションが無いので
最小限のドライバを使います。

```sh
apk add mesa-dri-gallium
```

---

### ■ 7. 再起動してデスクトップへ

```sh
reboot
```

再起動後、ログイン画面が表示され、LXQtを選択して使用できます。
